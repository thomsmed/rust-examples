# Animating sprites using AnimationPlayer

An example Bevy application using Bevy's [AnimationPlayer](https://docs.rs/bevy/latest/bevy/prelude/struct.AnimationPlayer.html) to animate a [Sprite](https://docs.rs/bevy/latest/bevy/sprite/struct.Sprite.html).

All the magic happens in [animated_sprite.rs](src/animated_sprite.rs).
